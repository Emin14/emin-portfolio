## Run Locally

Clone the project

```bash
  git clone https://github.com/Emin14/emin-portfolio.git
```

Go to the project directory

```bash
  cd emin-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Features

- Selecting a dark and light theme
